# Exercise 3 - Onboard a new employee in SAP SuccessFactors

In this exercise, you will onboard the new employee in the scenario with the name Jane Smit&lt;`NNN`&gt;. `NNN` will be replaced with your seat number, for example `Jane Smith045`.


| Step   | Description                | Screenshot |
| :----- | :------------------------- | :--------- |
| 3.1    | [Login to SuccessFactors](https://hcm-eu10-sales.hr.cloud.sap/login?company=SFLAP062575) as the HR administrator user `admin`.  |<a href="./images/3-1.jpg" target="_blank"><img src="./images/3-1.jpg" width="250"/></a>|
| 3.2    | In the Admin Center, start typing `Add New Employee` in the search bar.<br><br><b>Select</b> the action <b>Add New Employee</b> from the list.|<a href="./images/3-2.jpg" target="_blank"><img src="./images/3-2.jpg" width="250"/></a>|
| 3.3    | Enter the follwing values. and replace `NNN` with your seat number:<br><br><ul><li><b>Hire Date</b>: keep the default for today</li><li><b>Company</b>: Ace Germany</li><li><b>Event Reason</b>: New Hire (HIRNEW)</li><li><b>First Name</b>: Jane</li><li><b>Last Name</b>: Smith&lt;`NNN`&gt;</li><li><b>Display Name</b>: Jane Smith&lt;`NNN`&gt;</li></ul>|<a href="./images/3-3.jpg" target="_blank"><img src="./images/3-3.jpg" width="250"/></a>|
| 3.4    | <b>Scroll</b> down and enter the <b>Person Id</b>: jsmith&lt;`NNN`&gt;.<br><br>Replace `NNN` with your seat number.<br><br>Click <b>Continue</b>|<a href="./images/3-4.jpg" target="_blank"><img src="./images/3-4.jpg" width="250"/></a>|
| 3.5    | <b>Scroll</b> down and click <b>Continue</b>|<a href="./images/3-5.jpg" target="_blank"><img src="./images/3-5.jpg" width="250"/></a>|
| 3.6    | Select a value for <b>Job Classification<b> from the list, for example `Account Manager (ACC-MGR)`. |<a href="./images/3-6.jpg" target="_blank"><img src="./images/3-6.jpg" width="250"/></a>|
| 3.7    | <b>Scroll</b> down and click <b>Continue</b>|<a href="./images/3-7.jpg" target="_blank"><img src="./images/3-7.jpg" width="250"/></a>|
| 3.8    | <b>Click</b> on <b>Submit</b>.|<a href="./images/3-8.jpg" target="_blank"><img src="./images/3-8.jpg" width="250"/></a>|
| 3.9    | <b>Click</b> the link <b>View Profile of Jane Smith&lt;`NNN`&gt;</b>.|<a href="./images/3-9.jpg" target="_blank"><img src="./images/3-9.jpg" width="250"/></a>|
| 3.10   | Remeber the person Id <b>jsmith&lt;`NNN`&gt;</b> of the new employee.<br><br>You will need it in the next exercise.|<a href="./images/3-10.jpg" target="_blank"><img src="./images/3-10.jpg" width="250"/></a>|
